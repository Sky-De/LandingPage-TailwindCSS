## Installation

Provided step-by-step instructions on how to install and run this project locally.

```bash
# Clone the repository
git clone https://github.com/Sky-De/LandingPage-TailwindCSS

# Change directory
cd your-repo-name

# Install dependencies
npm install

# Run the project
npm run dev
```
